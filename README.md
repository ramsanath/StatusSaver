# Whatsapp Status Saver

A mobile application built on React Native to view, save and share WhatsApp statuses.

# Running It

    $ git clone https://github.com/ramsanath/StatusSaver
    $ cd StatusSaver
    $ npm install 
    $ ./lib_changes/copy.sh
    $ react-native run-android
